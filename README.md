# timofei-bazhukov-portfolio

## Getting started

### Start local
1. Create `.env.development.local` file with content `PUBLIC_URL=/`
2. ``` npm run start ```

### Deploy to github pages
``` npm run deploy ```

### abap2ui5 standalone

#### Start local (express server)
```
npm run abap2ui5-start-local
```
#### Start local (webpack)
```
npm run abap2ui5-start-webpack
```


### Flutter standalone

#### Start local (express server)
```
npm run flutter-start-local
```
