## Getting started

### Start local
``` 
Create `.env.development.local` file with content:
PUBLIC_URL=/
``` 
``` 
npm run start 
```

### Deploy to github pages
``` 
npm run deploy 
```

### abap2ui5 standalone

#### Start local (express server)
```
npm run abap2ui5-start-local
```
#### Start local (webpack)
```
npm run abap2ui5-start-webpack
```


### Flutter standalone

#### Start local (express server)
```
npm run flutter-start-local
```


### Tests

```
npm run test
```
